

# 🦁 Unsupervised Animal Classification


## 📁 Структура проекта

* `train.py` — обучает модель
* `predict.py` — предсказания через терминал
* `app.py` — веб-интерфейс
* `templates/index.html` — страница сайта
* `data/dataset/` — картинки животных
* `outputs/` — графики
* `uploads/` — загруженные картинки
* `.joblib` файлы — сохранённые модели
* `metadata.json` — информация о модели

---

## 🚀 Как запустить

### 1. Установка

```bash
pip install -r requirements.txt
```

---

### 2. Обучение

```bash
python train.py
```

👉 что происходит:

* загружаются картинки
* извлекаются признаки через ResNet50
* уменьшается размерность через UMAP
* данные делятся на группы (KMeans)
* считается качество (silhouette score)
* всё сохраняется

---

### 3. Предсказание (CLI)

```bash
python predict.py
```

👉 вводишь путь к картинке → получаешь кластер

---

### 4. Веб-интерфейс

```bash
python app.py
```

👉 открыть:
[http://localhost:5000](http://localhost:5000)

там можно:

* загрузить картинку
* увидеть кластер
* посмотреть график

---

## 🧠 Как работает модель

```
Картинка
  ↓
ResNet50 (признаки)
  ↓
UMAP (сжатие)
  ↓
KMeans (группы)
  ↓
Кластер (0–4)
```

---

## 💡 Важно

* модель **не знает, где кошка или собака**
* она просто группирует **похожие картинки**

---

## 📊 Метрика

* **Silhouette Score** от -1 до 1
* > 0.84 → норм результат

---

## ⚙️ Настройки

в `train.py`:

```python
NUM_CLUSTERS = 5
IMG_SIZE = (128, 128)
```
